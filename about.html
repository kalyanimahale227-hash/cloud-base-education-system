Cloud Based Education System

This Cloud Based Education System provides an online platform for students and teachers to manage courses, projects and learning materials using cloud technology.

Project Objective

The objective of this system is to provide centralized learning, online project management, secure authentication and cloud storage facilities for students and administrators.

System Modules

  • Student Registration and Login
  • Admin Dashboard
  • Project Upload and Management
  • Student Enrollment
  • Cloud File Storage

Technology Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python Flask
  • Database: In-Memory / DynamoDB
  • Cloud: AWS EC2, S3, IAM

Advantages

  • Remote access from anywhere
  • Secure authentication
  • Scalable cloud resources
  • Centralized project management
